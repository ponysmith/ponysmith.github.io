Callbacks

pinnd.js includes three callback functions which you can hook into by setting the appropriate functions in your options object on instantiation. There are callbacks for:

  • onpintop() : triggered when the element is pinned to the top of the viewport,
  • onpinbottom() : triggered when the element is pinned to the bottom of the context element (if defined),
  • onunpin() : triggered when the element is unpinned and returned to its original position

All three callback functions pass a jQuery reference to the pinned element(s) - the same one that was originally passed into pinnd() to instantiate the plugin.

In the example below, we've set callback functions to change the text of the pinned element to reflect its state:

Code for this example

{% highlight javascript linenos %} var options = { context: '#pinnd-context', offset_top: 50, offset_bottom: 10, onpintop: function(e) { e.html('Pinned to top'); }, onpinbottom: function(e) { e.html('Pinned to bottom'); }, onunpin: function(e) { e.html('Unpinned'); } } pinnd($('.pinnd-example'), options); {% endhighlight %}
