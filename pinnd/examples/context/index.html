Using Context

The context option in pinnd.js can be used to constrain the pinning behavior to a specific context. In the example below, scrolling down will pin our content to the top of the page as in the Basic Usage example. However, by using a context, we've constrained the pinning to the yellow box, so that when the pinnd element reaches the bottom of that box, it will pin to the bottom and scroll off screen. Scroll the page to give it a try.

Code for this example

{% highlight javascript linenos %} var options = { context: '#pinnd-context', offset_top: 50, offset_bottom: 10 } pinnd($('.pinnd-example'), options); {% endhighlight %}
