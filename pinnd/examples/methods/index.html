Public Methods

pinnd.js exposes two public methods, on() and off(). The public methods are stored in the object returned when instantiating the plugin with pinnd()

  • on() : enables pinning behavior on the elements. Note that unless you have set the autoenable option to false, on() is automatically called upon instantiation.
  • off() : disables pinning behavior on the elements.

In the example below, we've added a button to toggle the pinning behavior on and off. Click on the button and scroll to see the methods in action.

Code for this example

{% highlight javascript linenos %} // Create the pinnd element var options = { context: '#pinnd-context', offset_top: 50, offset_bottom: 10, } // Store the return value in a variable // We'll use this variable below to access our public methods var pinnd_example = pinnd($('.pinnd-example'), options); // Wire up the click behavior of the enable/disable link $('#pinnd-toggle').on('click', function(e) { e.preventDefault(); if($(this).hasClass('off')) { // Call the on() public method to enable pinning pinnd_example.on(); $(this).html('Enabled. Click to disable').addClass('on').removeClass('off'); } else if($(this).hasClass('on')) { // Call the off() public method to disable pinning pinnd_example.off(); $(this).html('Disabled. Click to enable').addClass('off').removeClass('on');; } }); {% endhighlight %}
Enabled. Click to disable.
