Basic example

This is the simplest example of using pinnd.js. As you scroll down the page, the box will pin to the top of the page (with a small offset to make sure the site's header bar doesn't get covered). Scrolling back up will unpin the box when it hits its original position. We've added some additional dummy text just to make sure we have room to scroll

This is our test object

Code for this example

{% highlight javascript linenos %} var options = { context: '#page-content', offset: 50 } var p = pinnd($('.pinnd-example'), options); {% endhighlight %}
